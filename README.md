# Greymind Dummy Key Generator

A tool for generating dummy keys of various formats

## Installation

``` bash
npm install
```

## Usage

``` bash
node ./index.mjs --help
node ./index.mjs --keys 20 --tag fifa-nl
node ./index.mjs --web
```

## Web query params

``` none
?keys=<keys>&format=<format>&tag=<tag>&download=<true|false>
```
